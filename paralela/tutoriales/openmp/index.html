Tutorial de OpenMP

Sean bienvenidos a un tutorial de OpenMP. El contenido de esta página, se divide en las siguientes secciones:


INSTALACIÓN

Se explica cómo instalar OpenMP en los sistemas operativos Windows y Linux (Para la distribución Ubuntu 14.04).

REPASO

Se introduce la definición de OpenMP, su modelo de construcción y un recorrido sobre algunos de los elementos característicos de OpenMP.

TUTORIAL

Se muestran unos ejemplos a nivel básico, intermedio y avanzado, donde se aplican elementos característicos de OpenMP; se explican detalles de implementación, análisis a nivel de programación secuencial y paralela, y un análisis de resultados en la ejecución de estos programas.

EJERCICIOS

Se propone una serie de ejercicios, donde se aplican elementos característicos de OpenMP, y estos ejercicios se basan en los ejemplos mostrados en el tutorial.
